# Netease-Downloader

一个简单的网易云歌单下载器。

## 准备环境

我这里用到了 Python3 以及 Requests 网络请求库。

Python3的安装可以参考我以前的文章，Requests安装命令：
```
$ pip3 install requests
```

## 运行脚本

```
git clone https://github.com/ec50n9/Netease-Downloader.git
cd Netease-Downloader
python3 main.py
# 然后根据提示输入你想要下载的歌单id就行。
```
